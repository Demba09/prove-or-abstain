# Product Hunt Post — Prove-or-Abstain

## Tagline
An AI agent that investigates product metric changes — and refuses to guess when the data doesn't prove a cause.

---

## Section 1: One-liner (158 chars)
Prove-or-Abstain: the AI analytics agent that only acts when statistics prove it — and abstains with a named reason when they don't.

---

## Section 2: Description (2-3 sentences)
Most analytics tools always give you an answer. Prove-or-Abstain is different: built on LangGraph, it applies strict statistical gates before asserting any root cause — and will refuse to act with a named reason (diffuse, not significant, entangled mechanism) when evidence is insufficient.

It handles SQL databases, Google Sheets, and CSV uploads. Deploy it in minutes.

---

## Section 3: Why (2-3 bullets)
- **No more hallucinated root causes** — four gates (material, localized, significant, clean) must ALL pass before the agent recommends an action
- **Real statistics, not vibes** — two-proportion z-test on the leading segment, p ≤ 0.01; exact rate/mix/interaction decomposition with zero residual
- **The ABSTAIN verdict is a feature** — an agent that can act needs a principled way to refuse. Prove-or-Abstain never fabricates a diagnosis
